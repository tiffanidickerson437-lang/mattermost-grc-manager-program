# generated

What the [engine](https://github.com/tiffanidickerson437-lang/compliance-program) rendered from [`companies/mattermost.config.yaml`](../companies/mattermost.config.yaml). Files here are outputs; edit the config and re-render rather than editing these by hand.

| File | What it is |
|---|---|
| [`in-scope-controls.yaml`](in-scope-controls.yaml) | The 45 controls the config put in scope, each carrying the reason it is in scope and the frameworks it satisfies |
| [`profile-selection.yaml`](profile-selection.yaml) | Which engine profiles the config selected and why |
| [`companies/mattermost/data.json`](companies/mattermost/data.json) | The fully rendered program: pillars, controls with Mattermost-specific narratives, and the roadmap |

This folder is also where the engine's pillars 01 through 03 (risk management, controls, TPRM) live in rendered form, which is why those numbers have no standalone folders in this repo.
