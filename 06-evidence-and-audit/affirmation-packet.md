### Why the gate watches the regulation too

CMMC Phase 2 was suspended 2026-07-13 and a 60-day Reform Task Force reports to the DoW CIO around mid-September 2026. Suspended is not repealed: Phase 1 self-assessment stands, DFARS 252.204-7012 is untouched, and 170.22 still demands an annual signature. Whatever the task force changes lands as an amendment to 32 CFR 170, which eCFR publishes with a date. So this gate reads the rule on every run and blocks when it moves. An affirmation against a superseded rule is the quiet version of a false claim.
